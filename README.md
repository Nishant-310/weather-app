🌦️ Weather App

A simple and responsive weather application that shows real-time weather data for selected cities using a public API.

🚀 Features
🌍 Select city from dropdown
🌡️ Displays real-time temperature
🌤️ Shows current weather condition
🎨 Dynamic background based on weather
⚡ Fast and responsive UI
🛠️ Tech Stack
HTML – Structure
CSS – Styling & Layout
JavaScript – Logic & API handling
OpenWeather API – Real-time weather data
